# Useful Chrome Extensions

This document lists recommended Chrome extensions that enhance developer productivity, security, and overall browsing
experience.

---

## 🛠️ Development Tools

- **React Developer Tools**  
  Inspect and debug the React component tree directly in Chrome DevTools.

- **Web Developer**  
  Adds a toolbar button with various web developer tools like CSS/HTML inspection, disabling styles, and more.

- **Lighthouse**  
  Auditing, performance metrics, accessibility, and best practices for web apps.

- **JSONVue**  
  Formats and highlights JSON responses for easier reading.

- **Multi Elasticsearch Heads**  
  Manage Elasticsearch clusters directly in the browser.

---

## 🔐 Security & Authentication

- **Authenticator**  
  Generate 2FA codes directly inside your browser.

- **LastPass**  
  Password manager to securely store and autofill credentials.

- **ModHeader**  
  Modify HTTP request headers to test APIs and web apps.

---

## 🍪 Cookies & Cache

- **EditThisCookie**  
  Manage, add, delete, and edit browser cookies with ease.

- **Clear Cache**  
  Quickly clear browser cache without closing tabs or reopening Chrome.

---

## 📖 Productivity & Reading

- **daily.dev**  
  A personalized developer news feed as your Chrome homepage.

- **Postlight Reader**  
  Strip clutter and render clean, readable versions of articles.

- **Selectext**  
  Copy text directly from YouTube videos or other videos with text embedded.

- **Otter.ai**  
  Record and transcribe meetings (Google Meet & Web Audio).

---

## 🌍 Translation Tools

- **Google Translate**  
  Instantly translate entire web pages and selected text.

- **TransOver**  
  Hover or click to translate words/phrases without leaving the page.

---

## 🚫 Ad & Content Blocking

- **Adblock**  
  Blocks ads and popups across websites.

- **uBlock Origin**  
  Efficient, lightweight ad blocker that uses fewer CPU and memory resources.
