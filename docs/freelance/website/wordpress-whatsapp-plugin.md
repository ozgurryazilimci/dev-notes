# WordPress WhatsApp Integration with Join.chat Plugin

This guide explains how to install, configure, and use the **Join.chat – WhatsApp plugin** to add a WhatsApp chat button
to your WordPress site.

---

## Introduction

Join.chat is a popular WordPress plugin that integrates **WhatsApp chat** directly into your website. It allows visitors
to quickly contact you via WhatsApp with a customizable chat button.

---

## Installation

1. Log in to your WordPress **Admin Dashboard**.
2. Go to **Plugins > Add New**.
3. In the search bar, type:  
   `Join.chat`
4. Locate **Join.chat – WhatsApp Chat Plugin** by Creame.
5. Click **Install Now**.
6. After installation, click **Activate**.

---

## Initial Setup

After activation:

1. Navigate to **Settings > Join.chat**.
2. Enter your **WhatsApp phone number** (include country code, e.g., `+90XXXXXXXXXX`).
3. Customize the default **chat message** that users will send when clicking the button.
4. Choose whether the chat opens **in WhatsApp Web (desktop)** or the **mobile app**.

---

## Display Options

- **Position**: Choose where the WhatsApp button appears (bottom-right or bottom-left).
- **Schedule**: Define working hours; the button can be hidden outside of office hours.
- **Devices**: Show or hide the button on desktop or mobile.
- **Call-to-Action (CTA)**: Add a short message above the button to encourage users to click.

---

## Advanced Settings

- **Pre-filled Message**:  
  You can include dynamic placeholders like:  
  `{url}` – The current page URL  
  `{title}` – The current page title

- **WooCommerce Integration** (if installed):  
  Display WhatsApp button on product pages or cart page for direct customer support.

- **Custom CSS**: Adjust the style of the WhatsApp button to match your theme.

---

## Testing

1. Visit your website frontend.
2. Look for the **WhatsApp chat button** (floating icon).
3. Click the button – it should open WhatsApp Web (desktop) or the WhatsApp mobile app with your pre-filled message.

---

## Best Practices

- Use a **dedicated business number** for WhatsApp.
- Set clear **working hours** to manage expectations.
- Add a **friendly CTA message** to increase engagement.
- Combine with **WooCommerce** for e-commerce support.
