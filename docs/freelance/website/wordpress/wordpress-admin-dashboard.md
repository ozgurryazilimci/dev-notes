# WordPress Admin Dashboard

This document provides a general overview of the WordPress Admin Dashboard, covering key sections and settings.

![](<images/wordpress_admin_dashboard.webp>)

---

## Admin Profile Settings

Each user can manage their personal profile:

- **Site Language**: Choose the preferred language for the admin interface.
- **Display Name**: Define how your name will be shown publicly.
- **Toolbar**: Enable/disable the admin toolbar on the frontend.
- **Password**: Generate or update a new password.

---

## Dashboard (Home Screen)

The admin homepage provides:

- **Welcome Panel**: Quick links to create a new post, add a page, or customize the site.
- **At a Glance**: Overview of posts, pages, and comments.
- **Activity**: Recent posts and comments.
- **Quick Draft**: Draft a post directly.
- **WordPress News**: Updates from the WordPress community.

---

## Posts

Manage and create blog content:

- **All Posts**: View and filter existing posts.
- **Bulk Actions**: Apply batch edits for author, category, tags, or comment status.
- **Add New**: Create new blog posts with the editor.
- **Categories**: Organize posts into hierarchical groups.
- **Tags**: Add descriptive labels to posts.

### Categories

- Add, edit, and manage post categories.
- Categories can be nested hierarchically.
- Each category has a name, slug (URL-friendly name), and optional description.

### Bulk Actions for Posts

- **Bulk Category Assignments**: Apply categories to multiple posts.
- **Bulk Tagging**: Add or remove tags for multiple posts at once.
- **Bulk Author Assignment**: Reassign posts to a different author.
- **Comment Options**: Open or close comments for multiple posts.

---

## Pages

Pages differ from posts (static content like “About” or “Contact”):

- **All Pages**: Manage existing pages.
- **Add New**: Create a new page.
- **Editor Options**: Use blocks or custom HTML.

---

## Media

Manage images, videos, and files:

- **Library**: Browse uploaded files.
- **Add New**: Upload files manually.
- **Settings**:
    - Store uploads in month- and year-based folders: **Disabled**.

---

## Appearance (Themes)

- **Themes**: Install and activate themes.
- **Customize**: Modify theme options (colors, fonts, layout).
- **Widgets**: Add or manage sidebar and footer content.
- **Menus**: Create navigation menus.

---

## Plugins

- **Installed Plugins**: View active/inactive plugins.
- **Add New**: Search and install from the WordPress plugin repository.
- **Activate/Deactivate**: Control functionality.

---

## Users

### Admin User Settings

- **Admin Site Language**: Change interface language.
- **Display Name**: Control how the admin appears publicly.
- **Menu Toolbar**: Enable/disable the top toolbar.
- **Password**: Update credentials.

### User Management

- **Add New User**: Create new users with roles.
- **Roles**: Subscriber, Contributor, Author, Editor, Administrator.

---

## Settings

### General

- **Site Title & Tagline (Description)**: Define your site’s identity.
- **Language**: Set the site language.
- **Timezone**: Choose your region.
- **Date Format & Time Format**: Customize display.
- **Membership**: Allow or disallow anyone to register.

### Writing

- **Default Post Category**: Set a default category for new posts.
- **Default Post Format**: Choose a default format (Standard, Aside, Gallery, etc.).
- **Post via Email**: Configure email settings to publish posts via email (optional).
- **Update Services**: Notify services when you publish new content.

### Reading

- **Homepage Displays**: Use a static page instead of latest posts.
- Choose a **Homepage** and a **Posts Page** with static page option.

### Discussion

- **Comment Approval**: Comments must be manually approved (**Enabled**).
- Manage pingbacks, notifications, and avatars.

### Media

- **Organize uploads into month- and year-based folders**: **Disabled**.

### Permalinks

- **Post Name**:  
  Example: `https://yourdomain.com/example-post/`
- **Custom Structure**:  
  Useful for product websites, e.g. `/category/product`

---

## Summary

The WordPress Admin Dashboard centralizes everything needed to create, manage, and customize a WordPress site. From
profile management and content creation to advanced settings and plugins, understanding each section ensures efficient
workflow and professional site management.
