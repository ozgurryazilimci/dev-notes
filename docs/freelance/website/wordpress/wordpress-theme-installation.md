# WordPress Theme Installation

This guide explains how to install and set up WordPress themes, with examples using **Astra** and **Storefront**. Themes
control the look and feel of your site and often provide ready-to-use templates.

---

## Installing a Theme via WordPress Admin

1. **Login to WordPress Admin**  
   Navigate to `https://yourdomain.com/wp-admin` and log in with your admin credentials.

2. **Go to Themes Section**
    - In the left sidebar, go to **Appearance → Themes**.
    - Click **Add New** to browse available themes.

You can view themes [here](https://wordpress.org/themes/) or search directly in the admin panel.
These are some recommended themes for you:

- [Astra](themes/astra-theme.md)
- [Cenote](themes/cenote-theme.md)
- [Tyche](themes/tyche-theme.md)
- [Storefront](themes/storefront-theme.md)

---

## Uploading a Theme Manually

If you have a `.zip` theme file (premium or downloaded externally):

1. Go to **Appearance → Themes → Add New → Upload Theme**.
2. Choose your `.zip` file and click **Install Now**.
3. Activate the theme once installation is complete.
4. Customize using the **Customize** menu or theme-specific options.

---

## Tips for Working with Themes

- **Child Theme**: Always consider creating a **child theme** if you plan to modify theme files to prevent losing
  changes during updates.
- **Starter Templates**: Many themes provide ready-to-use templates for quick setup.
- **Block Editor**: Most modern themes support Gutenberg (block editor) for easier content design.
- **Performance**: Lightweight themes like Astra improve page load speed and SEO.
