# WordPress Theme: Storefront

Storefront is a free theme designed specifically for WooCommerce stores.

![](<images/storefront_theme.webp>)

**Steps to Install and Use Storefront:**

1. **Search and Install**
    - In **Add Themes**, type **Storefront**.
    - Click **Install**, then **Activate**.

2. **WooCommerce Integration**
    - Storefront works seamlessly with WooCommerce.
    - After activation, follow WooCommerce setup wizard for your online store.

3. **Customize Storefront**
    - Go to **Appearance → Customize**.
    - Adjust layout, colors, typography, and homepage sections.
    - Use the block editor to modify pages and product listings.
