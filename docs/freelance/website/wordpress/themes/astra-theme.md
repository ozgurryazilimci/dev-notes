# WordPress Theme: Astra

Astra is a lightweight, customizable theme suitable for any type of website.

![](<images/astra_theme.webp>)

**Steps to Install and Use Astra:**

1. **Search and Install**
    - In **Add Themes**, type **Astra**.
    - Click **Install**, then **Activate**.

2. **Start Customizing**
    - Click **Appearance → Astra Options** or **Appearance → Customize**.
    - Choose **"Let's Get Started"** to begin setup.

3. **Build Your Website**
    - Astra provides **pre-built starter templates**.
    - Select **"Block Editor"** to edit content using WordPress’s Gutenberg editor.
    - Pick a **ready-made template** to quickly build your website.

4. **Customization Options**
    - Colors, typography, header/footer layouts.
    - Customize templates or start from scratch.
