# WordPress Theme: Cenote

Cenote is a modern and clean WordPress theme designed specifically for **blogging**. It comes with multiple starter
layouts suitable for personal, fashion, travel, or lifestyle blogs.

![](<images/cenote_theme.webp>)

**Steps to Install and Use Cenote:**

1. **Search and Install**
    - In **Add Themes**, type **Cenote** in the search bar.
    - Click **Install**, then **Activate**.

2. **Set Up Blog Layout**
    - Go to **Appearance → Customize**.
    - Under **Homepage Settings**, choose **Your latest posts** to display blog posts on the homepage.
    - Select a blog layout (grid, list, or full-width) from Cenote’s customization options.

3. **Customize the Theme**
    - **Fonts & Typography**: Adjust heading and body fonts for readability.
    - **Colors**: Choose your brand color palette.
    - **Header & Footer**: Configure navigation menus, logo, and social icons.

4. **Starter Sites (Optional)**
    - Install the **ThemeGrill Demo Importer** plugin.
    - Import a Cenote demo site (e.g., personal blog, lifestyle blog) for a quick start.

5. **Add Blog Posts**
    - Go to **Posts → Add New**.
    - Use the Block Editor to write and format your posts.
    - Add categories, tags, and featured images.

6. **Preview and Publish**
    - Review your changes in the live preview.
    - Publish once you’re ready.
